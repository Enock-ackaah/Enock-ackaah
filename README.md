## Current Research

### Statistical Reliability of Breast Cancer RNA-Sequencing Models

I am studying how development sample size affects predictive performance,
feature-selection stability, coefficient-direction reproducibility,
probability accuracy, and external generalization in machine-learning models
for estrogen-receptor status prediction.

The study uses:

- GSE81538 as the model-development cohort
- GSE96058 as the independent external-validation cohort
- 18,213 aligned RNA-sequencing features
- Repeated stratified validation
- Development-sample-size stress testing
- Feature-selection stability analysis
- Coefficient-direction reproducibility analysis
- Bootstrap confidence intervals
- External validation and Brier-score assessment
- Clean-environment model reproducibility verification

The project includes analysis notebooks, compact results, a reusable fitted
model, model documentation, SHA-256 artifact verification, and a complete
reproducibility package.

[View repository](https://github.com/Enock-ackaah/statistical-reliability-high-dimensional-er-models)
