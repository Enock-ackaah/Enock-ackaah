## Current Research

### Statistical Reliability of Breast Cancer RNA-Sequencing Models

I am investigating how development sample size affects predictive performance,
feature-selection stability, explanation reproducibility, probability error,
and external generalization in machine-learning models for estrogen-receptor
status prediction.

This research uses:

- GSE81538 as the model-development cohort
- GSE96058 as the independent external-validation cohort
- More than 18,000 aligned RNA-sequencing features
- Repeated stratified validation and sample-size stress testing
- Feature-selection stability analysis
- Coefficient-direction reproducibility analysis
- Bootstrap uncertainty estimation
- External validation and Brier-score assessment
